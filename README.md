Mesa17 Engine is a **Half-Life 2 inspired 3D engine** currently in development by Dragicornz. The project is designed for **iPadOS and Android**, with a focus on performance, modularity, and Source-style gameplay mechanics.  

## Planned Features

- Physics-based gameplay mechanics  
- Source-style level rendering and environments  
- Cross-platform support for iPadOS and Android  
- Touch and controller input  
- Modular architecture for custom assets  
- Play Test Build demo with `Gorden.gif` and `play.mp3`
